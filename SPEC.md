# 184-pinball — FC PINBALL

## 概要
ファミコン風ピンボールゲーム。単一 `index.html`、Canvas 2D、外部アセットなし。

## 操作
- 左フリッパー: ← / Z / 画面左タップ
- 右フリッパー: → / X / 画面右タップ
- 発射: Space / ↑ / LAUNCHボタン

## ゲームルール
- ボール3個。中央ドレイン（F-G間の赤破線ギャップ）に落ちたら1個消費
- ポップバンパー3個（100/100/150点）、スリングショット2個（各10点）
- スコア・ハイスコア（localStorage永続）はHUD上部にゼロ埋め表示
- 残ボールがなくなったらゲームオーバー、タップ/Spaceでリトライ

## 実装メモ
- 論理座標系 360×640（縦画面）。`resizeCanvas()` でDPR対応・アスペクト比維持のレターボックス
- 物理: 重力+壁/バンパー/スリングショット/フリッパーの反射。フリッパーは回転する線分（capsule）としてピボット角速度からボールへ運動量を伝達
- 描画は `docs/retro-style-guide.md` 準拠: `ctx.arc()` / `stroke()` によるアンチエイリアス曲線・斜め線を使わず、`pixCircle` / `pixLine` / `pixPolygonFill` で全てドット単位にスナップして描画
- BGM: 矩形波メロディ＋三角波ベース＋ノイズドラムの3声ループ（Web Audio、外部音源ファイルなし）
- iOS Safari対応: ダブルタップ防止・WebAudio unlock・pointerCapture仮想ボタン（`/ios-game-base` 準拠）

## 既知の割り切り
- プランジャーレーン（発射専用の別レーン）は実装せず、ボールは待機位置から直接上方向へ打ち出す簡易発射方式
- ティルト（台を揺らす操作）は未実装（スコープ外）
